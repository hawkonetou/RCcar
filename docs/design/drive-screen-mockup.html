▸ PRINCIPES DE LA REFONTE
CHIFFRE CENTRAL
Police Doto 900 weight, taille XXL. Glow cyan animé (2.6s) + aberration chromatique cyan/magenta. Crosshair brackets de cadrage HUD.
BADGE BATTERIE
Cellule stylisée + remplissage dégradé cyan→vert qui pulse 3.2s, hachures internes pour effet écran segmenté. Tension brute sous le %. Switch dégradé orange/rouge sous 20%.
ICÔNE DIAG ⌖
Bouton 26×26 px, glyphe target/réticule, opacité 60% au repos. Hover/tap → glow + outline net. Ouvre une bottom sheet plein-écran avec le journal diag.
SLIDER GRADUÉ
Track vertical avec dégradé bicolore (cyan haut → magenta bas), graduations 10%, ticks majeurs aux extrémités, marqueur central 0 mis en valeur, fill avec glow + bevel.
TÉLÉMÉTRIE LATÉRALE
3 cards à gauche : DIRECTION (LEDs triangulaires AV/AR), PWM brut /255, UPTIME. Renforce la sensation de console embarquée plutôt que d'app jouet.
CONNEXION EN COURS
Anneau double rotation contrarotative cyan + magenta, brackets de cadrage, mention SPP RFCOMM/UUID en sous-texte. Plus une transition qu'un état figé.
ATMOSPHÈRE
Grille blueprint cyan 28px + scanlines CRT animées (9s loop) + vignette + 2 radial-gradients diffus (cyan haut + magenta bas) pour la profondeur.
TYPOGRAPHIE
Doto 900 (chiffres / titres) — pixel grid moderne, signature visuelle. JetBrains Mono 300/400/600 (corps, télémétrie). Pas de Inter/Roboto.
ANIMATIONS
LED pulse 1.6s · battery pulse 3.2s · scanlines 9s · numglow 2.6s · ring spin 1.4s/2.2s · reveal staggered au boot (7 paliers 0.05-0.65s).
▸ PROCHAINE ÉTAPE
Port en Jetpack Compose : couleurs (Color.kt), composants (BatteryBadge, Slider redéfini, ConnectingScreen, telemetry cards), transitions (rememberInfiniteTransition pour les pulses, AnimatedVisibility pour reveal).
